# Feishu/Lark Integrations

整合飞书生态高星项目.

## Currently tracked

### 1. larksuite/cli (14.8K stars) - 官方 Lark CLI
- **Repo**: https://github.com/larksuite/cli
- **Why**: 官方工具, 我们应学习/依赖
- **Plan**: larkbotgo 集成 larksuite/cli 作为后端

### 2. chenhg5/cc-connect (13.2K stars) - AI agents -> Feishu
- **Repo**: https://github.com/chenhg5/cc-connect
- **Why**: Bridge Claude Code/Cursor/Gemini CLI to Feishu
- **Plan**: larkbotgo 加 "AI agent bridge" 模式 (类似 cc-connect)

### 3. langbot-app/LangBot (16.5K stars) - Agentic IM bots
- **Repo**: https://github.com/langbot-app/LangBot
- **Why**: Multi-platform agentic bots (vs 我们只 Lark)
- **Plan**: 加 Slack/Discord/Telegram 支持

### 4. ConnectAI-E/feishu-openai (5.6K stars) - Feishu + GPT-4
- **Repo**: https://github.com/ConnectAI-E/feishu-openai
- **Why**: 飞书 + 多模态 LLM (V/DALL-E/Whisper)
- **Plan**: 学习多模态集成模式
